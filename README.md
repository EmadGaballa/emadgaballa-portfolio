# 🚀 Local Development

Clone the repository

```bash
git clone https://github.com/EmadGaballa/emadgaballa-portfolio.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production version

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
